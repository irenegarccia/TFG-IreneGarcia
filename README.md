# Instrucciones para ejecutar el proyecto localmente

Para ejecutar el proyecto localmente, sigue los siguientes pasos:

1. **Actualizar los repositorios de tu sistema:**
   sudo apt update

2. **Instalar el módulo de entornos virtuales para Python:**
   sudo apt install python3-venv

3. **Instala el gestor de paquetes pip para Python3:**
   sudo apt install python3-pip

4. **Crear un entorno virtual con el siguiente comando:**
   python3 -m venv venv

5. **Activar el entorno virtual:**
   source venv/bin/activate

6. **Descargar proyecto:**
   sudo apt install git
   git clone https://github.com/irenegarccia/TFG-IreneGarcia

7. **Instalar las dependencias necesarias:**
   pip install -r requirements.txt

8. **Ejecutar aplicación:**
   python3 app.py
